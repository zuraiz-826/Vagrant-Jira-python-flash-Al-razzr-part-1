Lab 15: Using Templates with Conditionals & Loops
Objectives
Understand how to integrate Python data structures with HTML templates.
Learn to use Jinja2 loop constructs to dynamically generate content.
Implement Jinja2 conditional statements to customize content display based on criteria.
Prerequisites
Basic understanding of Python programming.
Familiarity with HTML structure and syntax.
Jinja2 templating knowledge is helpful but not necessary.
Introduction
Templates allow developers to populate static files (like HTML) with dynamic data via code. This lab will introduce using Jinja2 templates with conditionals and loops to create dynamic web content based on Python data structures.

Lab Tasks
Task 1: Pass a Python List or Dictionary to a Template
Set up your environment:

Ensure Python and Flask (or any preferred web framework that supports Jinja2) are installed.
Create a new project directory and navigate into it.
mkdir templates_lab
cd templates_lab
Create a Flask application:

Initialize a Flask app in app.py.
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    products = [
        {'name': 'Laptop', 'price': 999.99},
        {'name': 'Smartphone', 'price': 499.99},
        {'name': 'Tablet', 'price': 299.99}
    ]
    return render_template('index.html', products=products)

if __name__ == '__main__':
    app.run(debug=True)
Create an HTML template:

In the project directory, create a folder named templates.
Inside the templates folder, create a file named index.html.
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Product List</title>
</head>
<body>
    <h1>Product List</h1>
    <table border="1">
        <tr>
            <th>Name</th>
            <th>Price</th>
        </tr>
        {% for product in products %}
        <tr>
            <td>{{ product.name }}</td>
            <td>${{ product.price }}</td>
        </tr>
        {% endfor %}
    </table>
</body>
</html>
Task 2: Use a Jinja2 Loop to Display Items in a Table
Focus on the <tr> element inside the {% for %} loop. This loop iterates over the products list, dynamically generating a row in the table for each product.
Task 3: Use a Jinja2 Conditional Statement
Modify your template:

Add a conditional inside the loop to highlight products above a specific price.
<tr {% if product.price > 500 %}style="background-color: #f99;"{% endif %}>
    <td>{{ product.name }}</td>
    <td>${{ product.price }}</td>
</tr>
This if statement checks the product's price and applies a background color if it exceeds $500.
Task 4: Test Your Application
Run the Flask application:

python app.py
View in browser:

Open a web browser and navigate to http://127.0.0.1:5000/.
You should see a table of products with items priced above $500 highlighted.
Conclusion
You have successfully utilized Jinja2 templates with Python data structures to dynamically render HTML content. This lab demonstrated passing data to templates, utilizing loops for item display, and conditionally highlighting items directly within a Jinja2 template. These skills are fundamental for creating dynamic and interactive web pages.
